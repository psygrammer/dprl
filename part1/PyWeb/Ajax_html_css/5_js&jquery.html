2. JQuery

  • 자바스크립트를 쉽게 사용할수 있게 하는 라이브러리
  • 자바스크립트로 만들어진 다양한 함수들이 있음
  • jquery사이트에서 다운로드
  • 선택자 -> .메소드1 -> .메소드2 -> .메소드3
  • 객체 선택방법 :
    1. 직접선택 : $("#id") , $(".class"), $("tag"), $("*")
    2. 인접관계선택 : $("--").children, $("--").parent
    3. 탐색선택 : $("--").odd(), $("--").even(), $("--").first(), $("--").nth-child(3)
    1. 속성 조작 메소드 : $("-").html("-"), $("-").attr("속성","값"), $("-").css("스타일 속성","값"), $("-").val("값")
    2. 객체편집 메소드 : $("-").append("추가할내용")
    3. 애니메이션 메소드 : $("-").hide("효과 속도","콜백 함수"), $("-").show("효과 속도","콜백 함수"),
